##  Objectif de l'application

Planexia est une application Android développée en Java permettant aux utilisateurs de :

- Créer un compte et se connecter
- Gérer des modules de révision
- Ajouter des objectifs avec dates limites
- Ajouter des tâches pour chaque objectif
- Suivre la progression
- Visualiser un planning jour / semaine
- (Optionnel) Notifications et fonctionnalités Premium

##  Technologies utilisées

- Android Studio

- SQLite 


##  Architecture du projet

L'application est organisée en plusieurs parties :

- Authentification (Login / Register)
- Dashboard
- Modules
- Objectifs
- Tasks
- Planning
- Profil

Base de données :
- User
- Module
- Objective
- Task
